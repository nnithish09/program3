✅ Experiment 1: Git Local & Remote Repository (Basic Commands)
Aim

To create local and remote repositories using Git and GitHub and apply basic Git commands.

Steps
Install Git
Download and install Git from official website.

Create a Folder

cd Desktop
mkdir DevOps
cd DevOps
Create Repository on GitHub
Go to GitHub
Create new repository → Name: Prog
Copy HTTPS URL

Clone Repository

git clone <repository-URL>
cd Prog

Initialize Repository

git init

Create File

nano first.txt
Write: Hello World

Configure Git

git config --global user.email "your_email@example.com"
git config --global user.name "Your Name"
git config --list

Add File

git add first.txt

Check Status

git status
Commit Changes
git commit -m "First commit"
Push to GitHub
git push -u origin main
View Logs
git log
Reset (Undo changes)
git reset --hard

✅ Experiment 4: Maven Project in CI/CD
Aim

To create a Maven project with dependencies for CI/CD.

Steps
Open Eclipse
Use Eclipse IDE
Create Maven Project
File → New → Maven Project
Select maven-archetype-quickstart
Set Details
Group ID: exp11
Artifact ID: exp11

Add Dependency

Open pom.xml
Add:
<dependency>
    <groupId>com.googlecode.json-simple</groupId>
    <artifactId>json-simple</artifactId>
    <version>1.1.1</version>
</dependency>

Create Java Class
Path: src/main/java
Class: Exp11.java
Write Code (JSON Parsing)
import java.io.FileReader;
import org.json.simple.JSONObject;
import org.json.simple.parser.JSONParser;

public class Exp11 {
    public static void main(String[] args) throws Exception {
        JSONParser parser = new JSONParser();
        Object obj = parser.parse(new FileReader("emp.json"));

        JSONObject jsonObject = (JSONObject) obj;

        String fname = (String) jsonObject.get("firstname");
        String lname = (String) jsonObject.get("lastname");

        System.out.println("First Name: " + fname);
        System.out.println("Last Name: " + lname);
    }
}

Create JSON File
File: emp.json
{
  "firstname": "Ayush",
  "lastname": "Thanthri"
}
Run Project
Right-click → Run as Java Application
